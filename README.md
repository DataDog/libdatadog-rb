# `libdatadog-rb`

`libdatadog-rb` packages the [`libdatadog` Rust library](https://github.com/DataDog/libdatadog) for Ruby as a [gem published on rubygems.org](https://rubygems.org/gems/libdatadog).

## Installation

`libdatadog` is typically pulled in automatically by the `datadog` gem.

## Usage

Direct usage is discouraged as its API is considered internal to Datadog.
